# 期货风险分析方法论

> 本文档供 risk-analyst 子 Agent 在做期货账户分析时参考。
> 期货与股票风险逻辑完全不同：期货有杠杆、保证金、强平、移仓，必须单独处理。

## 一、核心概念

### 合约价值
```
合约价值 = 当前价 × 合约乘数 × 手数
```
例：螺纹钢 RB 当前 3180，乘数 10（吨/手），5 手 → 3180×10×5 = 159,000 元

### 占用保证金
```
占用保证金 = 合约价值 × 保证金比例
```
保证金比例通常 8%-15%，意味着 ~7-12 倍杠杆。

### 浮动盈亏
```
多头浮盈 = (当前价 - 开仓价) × 乘数 × 手数
空头浮盈 = (开仓价 - 当前价) × 乘数 × 手数
```

## 二、保证金占用率（账户健康度核心指标）

```
保证金占用率 = 总占用保证金 / 账户动态权益
```

| 占用率 | 含义 |
|---|---|
| <60% | 安全，有充足缓冲 |
| 60-80% | 关注，行情波动需留意 |
| 80-90% | 警示，缓冲不足 |
| >90% | 危险，跳空/连续不利波动易触发强平 |

## 三、强平距离（最关键的实战指标）

强平（平仓线）发生在：**账户动态权益 ≤ 维持保证金**。

简化的"强平距离"估算（仅该合约反向波动、其他不变）：
```
账户缓冲 = 动态权益 - 占用保证金
该合约每点亏损 = 乘数 × 手数
最大可承受反向点数 = 账户缓冲 / 每点亏损
强平距离(%) = 最大可承受反向点数 / 当前价
```

解读：强平距离 = "这个合约还能反向波动多少 % 才会逼近追保线"。

| 强平距离 | 含义 |
|---|---|
| >10% | 关注 |
| 5-10% | 警示 |
| <5% | 危险（一个跌停 ~10% 就可能直接强平） |
| <3% | 极度危险 |

> 注意：这是近似估算。实际强平还受维持保证金率、跨品种盈亏对冲、保证金动态调整影响。
> 估算的价值在于让用户**直观感知"距离爆仓还有多远"**。

## 四、跨品种方向性集中（容易被忽视的风险）

### 单品种敞口
```
单品种敞口占比 = 该品种合约价值 / 总合约价值
```
单一品种占比过高（>50%）→ 该品种异动会重创账户。

### 方向性集中
```
净方向比率 = (多头合约价值 - 空头合约价值) / 总合约价值
```
- 接近 +1：几乎全是多头 → 市场整体下跌时全军覆没
- 接近 -1：几乎全是空头 → 市场整体上涨时全军覆没
- 接近 0：多空对冲，方向性风险低

### 板块同向风险（解读时重点提示）
典型陷阱：
- 黑色系全做多（螺纹+铁矿+焦炭+焦煤）→ 表面是 4 个品种，实质是 1 个方向的赌注
- 油脂全做多（豆油+菜油+棕榈油）
- 当板块同向波动时，"分散"是假象，实际风险高度集中

## 五、隔夜风险与移仓

- **隔夜风险**：夜盘 + 外盘联动（原油、贵金属、有色尤其），跳空可能直接击穿强平线
- **移仓风险**：临近交割（≤30天）的合约，主力切换有滑点、流动性下降、临近交割有逼仓风险 → 提示移仓

## 六、解读原则

- 用"还能反向波动 X% 就追保"这种**直观语言**，不要堆术语
- 重点提示：占用率过高、强平距离过近、方向性过度集中
- 永远不给"建议平仓 X 手"这类操作建议，只描述风险事实
