### 关键架构决策

- **框架**：LangGraph 直接构建（create_react_agent + Agent-as-Tool），不用 DeepAgents/沙箱
- **分工**：工具算数字（确定性）+ LLM 解读（数字→人话）
- **网关**：端口-适配器模式（借鉴 openclaw），ChannelPlugin 契约 + Gateway 调度，加平台零侵入

### 当前能力（已上线飞书）

> 在飞书里私聊机器人，发持仓数据 → 返回风险评估报告（集中度、VaR、期货保证金/强平）。

完整链路：飞书 → FeishuChannel → Gateway → Runtime → 主Agent → risk-analyst → 风险工具 → 大模型解读 → 合规回复 → 飞书。
